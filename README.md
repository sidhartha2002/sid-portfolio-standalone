# Sidhartha Mohanty — Portfolio

A premium animated personal portfolio built with React, Vite, TypeScript, Tailwind CSS, and Framer Motion.

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import the repo
3. Vercel will auto-detect Vite — just hit **Deploy**
4. Add your resume PDF: upload `Sidhartha_Mohanty_Resume_2026.pdf` to the `public/` folder

## Deploy to Netlify

1. Push to GitHub
2. Netlify → Add new site → Import from Git
3. Build command: `npm run build`
4. Publish directory: `dist`

## Adding Your Resume

Place your resume PDF at:
```
public/Sidhartha_Mohanty_Resume_2026.pdf
```
The Download Resume buttons already link to this path.
